# 电子包浆生成器 | Digital Patina Maker

> 一键生成互联网“电子包浆”效果，模拟图片经过多次转存、截图、压缩产生的岁月痕迹与画质劣损感。

## 🌟 特性 (Features)

- **纯前端实现**：无需后端服务器，所有图片处理（Canvas 渲染）均在本地浏览器中完成。
- **Web Worker 并行处理**：支持多张图片同时处理而不会引发主线程的卡顿。
- **预设「包浆」滤镜**：
  - 🎞️ **09年老图 (Mild)**: 轻度失真与泛黄，呈现早期相机的年代感。
  - 💬 **贴吧神图 (Medium)**: 经典的多次压缩变绿与噪点，复刻论坛时代的“极品老图”。
  - ☢️ **电子废土 (Heavy)**: 极度的像素化与色彩崩坏，终极赛博失真效果。
- **批量下载**：支持一键将处理完成的表情包打包为 ZIP 下载。
